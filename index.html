Webkit

the scrollbar can auto adjust when the content been dynamic changed.(e.g. nodes inside been removed or inserted)

IE

you need to call refresh() , when the content been dynamic changed.

Install 

$ bower install customize-scrollbar

If you need help about bower, see Bower's HomePage.

Dependency

  • jquery #1.x
  • jquery-mousewheel #3.x

Get Started

HTML

<div class="scrollbar">
    <div class="up"></div>
    <div class="thumb"></div>
    <div class="down"></div>
    <div class="demo">......</div>
</div>

CSS

all css can be customize

JS

$(function(){
    $('.demo').scrollbar({
        scrollbar : '.scrollbar',
        up        : '.up',
        thumb     : '.thumb',
        down      : '.down'
    });
});
